## 一、核心领域对象（Domain Objects）

### 1.1 标的 Symbol

系统内统一使用 **Futu 格式**作为主键，其他来源的代码在进入系统时做规范化转换。

| 来源 | 格式示例 | Futu 规范格式 |
|:---|:---|:---|
| Futu 港股 | `00700` | `HK.00700` |
| Futu 美股 | `AAPL` | `US.AAPL` |
| Futu A股 | `000001` | `SZ.000001` |
| YFinance | `AAPL` | `US.AAPL` |
| Finnhub | `AAPL` | `US.AAPL` |

```typescript
// TypeScript
type Market = 'US' | 'HK' | 'SH' | 'SZ' | 'CRYPTO'

interface Symbol {
  futu_code: string      // 主键，如 "US.AAPL"
  display_code: string   // 展示用，如 "AAPL" 或 "00700"
  name: string           // "Apple Inc." 或 "腾讯控股"
  market: Market
  currency: string       // "USD" | "HKD" | "CNY"
  lot_size: number       // 每手股数（港股 500，美股 1）
}
```

## 二、PostgreSQL Schema

### 2.1 核心表设计

```sql
-- 用户（个人系统，只有一条记录）
CREATE TABLE users (
  id           UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  password_hash VARCHAR(256) NOT NULL,
  created_at   TIMESTAMPTZ DEFAULT NOW()
);

-- 刷新 Token 黑名单
CREATE TABLE refresh_token_blacklist (
  jti          VARCHAR(64) PRIMARY KEY,  -- JWT ID
  expires_at   TIMESTAMPTZ NOT NULL,
  created_at   TIMESTAMPTZ DEFAULT NOW()
);
CREATE INDEX idx_rt_blacklist_expires ON refresh_token_blacklist(expires_at);

-- 订单记录（模拟 + 实盘都写入）
CREATE TABLE orders (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_id      VARCHAR(64) UNIQUE NOT NULL,  -- Futu 单号 or 本地模拟 ID
  symbol        VARCHAR(32) NOT NULL,
  side          VARCHAR(4) NOT NULL CHECK (side IN ('buy', 'sell')),
  order_type    VARCHAR(12) NOT NULL,
  qty           INTEGER NOT NULL,
  filled_qty    INTEGER DEFAULT 0,
  price         DECIMAL(18, 4),
  avg_fill_price DECIMAL(18, 4),
  status        VARCHAR(12) NOT NULL,
  is_simulated  BOOLEAN DEFAULT TRUE,
  note          TEXT,
  created_at    TIMESTAMPTZ DEFAULT NOW(),
  updated_at    TIMESTAMPTZ DEFAULT NOW()
);
CREATE INDEX idx_orders_symbol ON orders(symbol);
CREATE INDEX idx_orders_status ON orders(status);
CREATE INDEX idx_orders_created ON orders(created_at DESC);

-- 策略/研报 RAG 向量存储
CREATE EXTENSION IF NOT EXISTS vector;

CREATE TABLE knowledge_chunks (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  url         TEXT,                             -- 来源 URL（网页或文件路径）
  title       TEXT,
  content     TEXT NOT NULL,
  embedding   VECTOR(1536),                     -- OpenAI text-embedding-3-small 维度
  source_type VARCHAR(32),                      -- 'webpage' | 'report_pdf' | 'news'
  metadata    JSONB DEFAULT '{}',
  created_at  TIMESTAMPTZ DEFAULT NOW()
);
CREATE INDEX idx_knowledge_embedding ON knowledge_chunks USING ivfflat (embedding vector_cosine_ops);
CREATE INDEX idx_knowledge_url ON knowledge_chunks(url);
CREATE INDEX idx_knowledge_created ON knowledge_chunks(created_at DESC);

-- 操作审计日志
CREATE TABLE audit_logs (
  id         UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  action     VARCHAR(64) NOT NULL,   -- 'login' | 'order_simulate' | 'order_execute' | 'settings_change'
  detail     JSONB DEFAULT '{}',
  ip         VARCHAR(45),
  trace_id   VARCHAR(64),
  created_at TIMESTAMPTZ DEFAULT NOW()
);
CREATE INDEX idx_audit_action ON audit_logs(action);
CREATE INDEX idx_audit_created ON audit_logs(created_at DESC);

-- 客户端 APM 心跳（按天分区）
CREATE TABLE client_heartbeats (
  id           UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  platform     VARCHAR(16) NOT NULL,   -- 'ios' | 'android' | 'harmonyos'
  app_version  VARCHAR(32),
  device_id    VARCHAR(128),
  fps          DECIMAL(5, 1),
  memory_mb    DECIMAL(8, 1),
  ws_latency_ms INTEGER,
  ts           TIMESTAMPTZ DEFAULT NOW()
) PARTITION BY RANGE (ts);

CREATE TABLE client_heartbeats_2026 PARTITION OF client_heartbeats
  FOR VALUES FROM ('2026-01-01') TO ('2027-01-01');
```

## 三、Redis Key 命名规范

### 3.1 命名规则

```
{namespace}:{entity}:{identifier}[:{sub_key}]
```

| 前缀 | 含义 |
|:---|:---|
| `quote:` | 实时报价 |
| `kline:` | K线缓存 |
| `session:` | JWT 会话 |
| `rt:blacklist:` | Refresh Token 黑名单 |
| `screener:` | 选股结果缓存 |
| `tool:cache:` | Agent Tool 调用结果缓存 |
| `ws:conn:` | WebSocket 连接追踪 |
| `circuit:` | 熔断器状态 |

### 3.2 Key 目录

```
# 实时报价快照（Hash，TTL: 5min）
quote:snapshot:{futu_code}
  HSET price 213.45
  HSET change_pct 0.0058
  HSET ts 1719475200123
  HSET volume 58234901

# K线缓存（Hash，热数据，TTL: 24h）
kline:hot:{futu_code}:{period}
  HSET data <msgpack_bytes>
  HSET updated_at 1719475200000

# K线实时更新推送（Pub/Sub Channel）
kline:channel:{futu_code}:{period}

# 行情 Tick 推送（Pub/Sub Channel）
quote:channel:{futu_code}

# JWT Refresh Token 黑名单（Set，TTL 对齐 token 过期时间）
rt:blacklist:{jti}

# Agent Tool 结果缓存（String，TTL: 可配置）
tool:cache:{tool_name}:{args_hash}

# WebSocket 在线连接数（Gauge）
ws:conn:count

# 选股结果缓存（String / JSON，TTL: 10min）
screener:result:{query_hash}

# 熔断器状态（String，TTL: 60s）
# 值: "closed" | "open" | "half_open"
circuit:futu_api
circuit:yfinance_api
circuit:openai_api
```

## 四、数据流转示意

```
Futu OpenD (Raw Tick)
    │
    ▼ ZeroMQ PUSH
workers/quote_publisher.py
    │ 清洗 + 规范化 → QuoteModel
    ├──► Redis HSET quote:snapshot:{symbol}   ← REST API 读取（低频）
    └──► Redis PUBLISH quote:channel:{symbol}  ← WebSocket Worker 订阅（高频推送）
                                                    │
                                                    ▼
                                              前端 /ws/v1/quotes
                                              Flutter WS 客户端
```

```
前端请求历史 K 线
    │
    ▼ GET /api/v1/market/kline
MarketService.get_kline()
    │
    ├─ ① 检查 Redis kline:hot:{symbol}:{period}   → HIT: 直接返回（< 1ms）
    │
    ├─ ② 检查 DuckDB/Parquet 本地数据湖            → HIT: 返回并回填 Redis（< 100ms）
    │
    └─ ③ Futu API 拉取 → 写入 DuckDB → 回填 Redis → 返回（< 2000ms）
```
