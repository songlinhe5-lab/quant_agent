# 前端视觉设计规范（标杆映射）

> **任务**：FE-26 · **版本**：2026-07-13  
> **关联**：`frontend/src/styles/globals.css` · `frontend/src/lib/motion.ts` · `docs/04`

本文把 Linear / Vercel / Robinhood 的可借视觉特征，映射到 Quant Agent 的 CSS Token，避免「赛博朋克堆料」与「默认紫渐变」两极。

---

## 1. 标杆特征 → 本仓 Token

| 标杆 | 可借特征 | 本仓落地 |
|:---|:---|:---|
| **Linear** | 极低对比边框、高信噪比、微动效（150–200ms） | `--border` / `--motion-fast|base`；侧栏域色点 + 3px 激活条 |
| **Vercel** | 近黑底、白字层级、玻璃面板克制 | `.dark` 背景 `240 10% 3.9%`；`.glass-card` 半透明 + `border-white/10` |
| **Robinhood** | 涨跌语义色清晰、数字等宽、行情闪烁短促 | `--color-bull` / `--color-bear`；`tabular-nums`；`.tick-up/.tick-down` 400ms |

**禁止照搬**：Robinhood 大面积绿底营销页、Linear 纯白办公风作为默认主题、Vercel 营销站巨型字重。

---

## 2. 语义色（FE-25）

| 语义 | CSS 变量 | Tailwind |
|:---|:---|:---|
| 涨 / 多 / 盈 | `--color-bull` | `text-bull` / `text-emerald-400` |
| 跌 / 空 / 亏 | `--color-bear` | `text-bear` / `text-red-400` |
| 警告 / STALE | `--color-warn` | `text-warn` / `text-amber-500` |
| 信息 / 链接 | `--color-info` | `text-info` |

STALE：父级加 `.stale-data`（降不透明度 + 去饱和）+ `.stale-badge` 文案。

---

## 3. 动效与 Toast（FE-28）

见 `frontend/src/lib/motion.ts`：

- 微交互 `150ms` · 默认 `200ms` · 抽屉 `300ms` · 闪烁 `400ms`
- Toast 自动消失 `4500ms`，同屏上限 `3`

Lighthouse 基准：URL 加 `?lighthouse=1` 挂 `.reduce-motion`（FE-14）。

**2026-07-13 实测**（登录页，`pnpm lighthouse:baseline`）：

| Form factor | Performance | Accessibility | LCP | 备注 |
|:---|---:|---:|:---|:---|
| desktop（验收口径） | **95** | 100 | 1.1s | ≥85 通过 |
| mobile（模拟 4G） | 51 | 100 | 6.4s | 非主场景；unused JS ~594KiB 是主拖累 |

报告：`frontend/.lighthouse/baseline-desktop.report.html`
