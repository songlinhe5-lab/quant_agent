## 一、架构决策：Flutter 统一三端（ADR-002）

### 1.1 放弃原生四端，选用 Flutter

**放弃**：Tauri（桌面）+ Swift/SwiftUI（iOS）+ Kotlin/Jetpack Compose（Android）+ ArkTS/ArkUI（HarmonyOS）四套独立代码库  
**选用**：Flutter（Dart）统一覆盖 **Android + iOS + HarmonyOS NEXT** 三端

| 维度 | Flutter 三端（已选） | 原生三端（已放弃） |
|:---|:---|:---|
| **代码复用率** | 业务逻辑 ~90%、UI ~70% 共享 | 三套独立代码，Bug 修复重复三次 |
| **开发效率** | 单兵可维护 | 需 3+ 人分别维护 Swift / Kotlin / ArkTS |
| **渲染引擎** | Impeller（Vulkan/Metal 后端），60~120fps | 各端各自 GPU 方案，但开发成本极高 |
| **图表性能** | CustomPainter + RepaintBoundary，GPU 加速 | Metal / Vulkan / NAPI 极致，但工作量巨大 |
| **热重载** | ✅ 支持，开发效率极高 | ❌ 原生编译等待 |
| **推送通知** | APNs + FCM + HMS Push Kit 统一管理 | 三套推送证书与渠道独立维护 |
| **鸿蒙支持** | 华为官方 Flutter Fork（已支持 HarmonyOS NEXT）| ArkTS / ArkUI 原生，但生态尚未成熟 |
| **上架** | App Store + Google Play + 华为应用市场 同步构建 | 三个商店独立审核流程 |

**结论**：对单兵量化投研者，Flutter 是正确的工程经济学选择。鸿蒙端通过华为维护的 [flutter-harmonyos](https://gitee.com/openharmony-sig/flutter_samples) 分支支持，不另立代码库。

### 1.2 三端目标定位

| 端 | 定位 | 核心场景 |
|:---|:---|:---|
| **Android** | 随身监控 + 移动交易 | 通勤查持仓、接推送处理告警、AI 语音问盘 |
| **iOS** | 随身监控 + 移动交易 | 与 Android 功能对等，Face ID 鉴权，APNs 高优推送 |
| **HarmonyOS NEXT** | 随身监控 + 鸿蒙生态协同 | 华为设备全场景：手机 + 平板 + 手表告警 + 分布式流转 |

## 二、技术栈（零偏差约束）

```
Flutter SDK        : 3.22+（Dart 3.4+）
鸿蒙 Flutter Fork  : 华为官方分支 flutter-harmonyos（HarmonyOS NEXT 5.0+）
渲染后端           : Impeller（Android: Vulkan | iOS: Metal | HarmonyOS: Vulkan）
状态管理           : Riverpod 2.x（代码生成版 @riverpod，类型安全）
导航路由           : go_router 14.x（声明式路由 + Deep Link）
网络               : dart:io WebSocket（行情流）| http 1.x（REST + SSE）
本地存储           : Isar 3.x（高性能嵌入式 DB，K线历史缓存）
安全存储           : flutter_secure_storage（Keychain / Keystore / HarmonyOS Keystore）
推送通知           :
  iOS → APNs via firebase_messaging
  Android → FCM via firebase_messaging
  HarmonyOS → HMS Push Kit（独立接入，华为应用市场要求）
图表引擎           : 自研 CustomPainter + CanvasKit（K线主图）
APM / 崩溃         : firebase_crashlytics（iOS/Android）+ 自研 AppMonitor 心跳上报
```

**严禁**：
- ❌ WebView 内嵌 ECharts 渲染 K线（体验割裂，性能不可控）
- ❌ `fl_chart` 用于 K线主图（性能不足，无法 60fps 手势缩放）
- ❌ `SharedPreferences` 存 JWT Token（明文，安全漏洞）
- ❌ `setState` 管理跨组件高频行情状态（应使用 Riverpod StreamProvider）
- ❌ 主 Isolate 反序列化大量历史 K线 JSON（应 `compute()` 卸载）

```dart
// 华为折叠屏（如 Mate X6）：展开态 ≥ 700px → 双列布局

class HarmonyAdaptive extends StatelessWidget {

      : MobileLayout(child: child);

}
```

## 六、高频数据管道（Flutter 零 GC 设计）

### 6.1 WebSocket 客户端（断连指数退避重试）

```dart
// core/network/ws_client.dart

class WsClient {
  WebSocket? _ws;
  int _retryCount = 0;
  static const _maxRetry = 5;

  final _tickController = StreamController<TickData>.broadcast();
  Stream<TickData> get tickStream => _tickController.stream;

  Future<void> connect(String url, String token) async {
    try {
      _ws = await WebSocket.connect('$url?token=$token');
      _retryCount = 0;
      AppMonitor.instance.recordWsConnected();

      _ws!.listen(
        (data) {
          final tick = TickData.fromRawJson(data as String);
          _tickController.add(tick);
        },
        onDone: _handleDisconnect,
        onError: (e) { AppMonitor.instance.recordWsError(e); _handleDisconnect(); },
      );
    } catch (_) { _handleDisconnect(); }
  }

  void _handleDisconnect() {
    AppMonitor.instance.recordWsDisconnect();
    if (_retryCount < _maxRetry) {
      final delay = Duration(seconds: pow(2, _retryCount).toInt());  // 1/2/4/8/16s
      _retryCount++;
      Future.delayed(delay, () => connect(_lastUrl, _lastToken));
    }
  }
}
```

```dart
Stream<TickData> tickStream(TickStreamRef ref, String symbol) {
```

```dart
      next.whenData((tick) => setState(() => _updateLastKline(tick)));
    });

    return RepaintBoundary(                // 隔离重绘边界 — 关键！
      child: GestureDetector(
        onScaleUpdate: (d) => setState(() {
          _scaleX = (_scaleX * d.scale).clamp(0.2, 5.0);
          _offsetX += d.focalPointDelta.dx;
        }),
        onLongPressStart: (d) => setState(() => _crosshairPos = d.localPosition),
        onLongPressEnd:   (_) => setState(() => _crosshairPos = null),
        onDoubleTap: () => setState(() { _scaleX = 1.0; _offsetX = 0.0; }),
        child: CustomPaint(
          painter: KlinePainter(
            klines: _klines,
            scaleX: _scaleX, offsetX: _offsetX,
            crosshairPos: _crosshairPos,
            bullColor: AppColors.bull, bearColor: AppColors.bear,
          ),
          size: Size.infinite,
        ),
      ),
    );
  }
}
```

### 8.2 AppMonitor 采集指标

```dart
// core/monitoring/app_monitor.dart

class AppMonitor {
  static final instance = AppMonitor._();
  AppMonitor._();

  double _fps = 60.0;
  int _wsDisconnects = 0, _errorCount = 0;
  int _wsLatencyMs = 0;

  void startFpsMonitor() {
    SchedulerBinding.instance.addTimingsCallback((timings) {
      if (timings.isEmpty) return;
      final avgMs = timings.map((t) => t.totalSpan.inMicroseconds / 1000).reduce((a, b) => a + b) / timings.length;
      _fps = 1000.0 / avgMs;
    });
  }

  void recordWsConnected()         => _wsLatencyMs = DateTime.now().millisecondsSinceEpoch;
  void recordWsDisconnect()        => _wsDisconnects++;
  void recordWsError(Object e)     => _errorCount++;
  void recordWsLatency(int ms)     => _wsLatencyMs = ms;

  Future<void> sendHeartbeat(ApiClient api) async {
    await api.post('/client/heartbeat', {
      'client_id':     _clientId,
      'platform':      _detectPlatform(),  // 'android' | 'ios' | 'harmonyos'
      'app_version':   _appVersion,
      'fps_current':   _fps,
      'memory_mb':     await _getMemoryMb(),
      'ws_latency_ms': _wsLatencyMs,
      'ws_disconnects': _wsDisconnects,
      'error_count':   _errorCount,
      'active_screen': _currentRoute,
      'trading_mode':  _tradingMode,
      'timestamp':     DateTime.now().toIso8601String(),
    });
  }

  String _detectPlatform() {
    if (Platform.isAndroid) return 'android';
    if (Platform.isIOS)     return 'ios';
    // HarmonyOS：Android 内核，通过 ro.build.version.emui 区分
    return 'harmonyos';
  }
}
```

## 九、交互完整性规范

### 9.1 WebSocket 断连处理流程

```
WebSocket 断连事件触发
  ↓
1. 立即：所有行情 Widget → StaleOverlay 显示
   立即：AppBar 状态图标变红（or 顶部红色 Banner）
   立即：通知 AppMonitor 记录断连
   ↓
2. 指数退避重连（1s / 2s / 4s / 8s / 16s）
   展示：Banner "正在重连 (2/5)..."
   ↓
3. 重连成功：StaleOverlay 淡出（300ms transition）
   ↓
4. 5 次全失败：发送本地通知 "行情连接中断，请检查网络"
   App 内显示持久 Banner + [手动重连] 按钮
```

### 9.2 生命周期感知（防后台耗电）

```dart
// core/network/ws_lifecycle_observer.dart

class WsLifecycleObserver extends WidgetsBindingObserver {
  @override
  void didChangeAppLifecycleState(AppLifecycleState state) {
    switch (state) {
      case AppLifecycleState.paused:
        WsClient.instance.pauseMarketSubscription();  // 暂停行情订阅
        AppMonitor.instance.recordBackground();
      case AppLifecycleState.detached:
        WsClient.instance.disconnect();               // 彻底关闭 WS
      case AppLifecycleState.resumed:
        WsClient.instance.reconnect();                // 重新订阅
        AppMonitor.instance.recordForeground();
      default: break;
    }
  }
}
```
